November 2017

Age Ordering Algorithm

Created an algorithm in Python to order varying nuumbers of wizards based on age. The wizards are presented in a list of constraints if the following structure: (a b c), which corresponds to wizard c not being in between wizard a and wizard b in age. The number of constraints given is the number of wizards that need to be ordered. A variation of the 3-SAT algorithm was used and we used the 3-Change technique to get out of local minima.
