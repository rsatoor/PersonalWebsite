Age Ordering Algorithm

November 2017

Created an algorithm in Python to order varying nuumbers of wizards based on age. The wizards are presented in a list of constraints in the following structure: [A B C], which corresponds to wizard C not being in between wizard A and wizard B in age. The number of constraints given is the number of wizards that need to be ordered. A variation of the 3-SAT algorithm was used and I used the 3-Change technique to get out of local minima.
